# Basic CRUD Application using PHP & MySQL

This project demonstrates a basic CRUD web application with user authentication.

## Features
- User Registration and Login
- Password Hashing
- Session Management
- Create, Read, Update, Delete Posts

## Tech Stack
- PHP
- MySQL
- XAMPP
- HTML

## Run Instructions
1. Install XAMPP
2. Place project in htdocs
3. Create database `blog`
4. Create tables `users` and `posts`
5. Open http://localhost/crud-app/login.php
